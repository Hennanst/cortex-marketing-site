Política de Privacidade

Última atualização: 05/04/2026

1. Informações que Coletamos

O Setup Cortex coleta informações limitadas para melhorar sua experiência:

  • Dados de navegação: páginas visitadas, tempo de permanência, tipo de dispositivo e navegador
  • Cookies: utilizamos cookies para análise de tráfego (Google Analytics) e otimização de anúncios (Google Ads)
  • Informações de cliques: registramos cliques em links de produtos para fins de análise de performance

2. Como Usamos as Informações

  • Melhorar o conteúdo e recomendações do site
  • Analisar o desempenho das páginas e produtos
  • Personalizar a experiência do usuário
  • Otimizar campanhas de marketing

3. Cookies e Tecnologias de Rastreamento

Utilizamos as seguintes tecnologias:

  • Google Analytics (GA4): para análise de tráfego e comportamento dos visitantes
  • Google Ads: para mensuração de conversões e remarketing
  • Facebook Pixel: para otimização de anúncios nas redes sociais
  • Amazon Associates: para rastreamento de compras realizadas através dos nossos links de afiliado

Você pode desativar cookies através das configurações do seu navegador. Note que isso pode afetar a funcionalidade do site.

4. Links de Afiliados

Este site participa do Programa de Associados da Amazon. Links para produtos na Amazon contêm nosso identificador de afiliado. Quando você realiza uma compra através desses links, recebemos uma comissão sem custo adicional para você.

5. Compartilhamento de Dados

Não vendemos, trocamos ou transferimos suas informações pessoais para terceiros. Os dados coletados são compartilhados apenas com:

  • Google (Analytics e Ads) — para fins de análise
  • Amazon — para rastreamento de vendas por afiliado

6. Segurança

Implementamos medidas de segurança adequadas para proteger contra acesso não autorizado, alteração, divulgação ou destruição de dados.

7. Seus Direitos (LGPD)

De acordo com a Lei Geral de Proteção de Dados (LGPD), você tem direito a:

  • Acessar seus dados pessoais
  • Solicitar correção de dados incompletos
  • Solicitar exclusão de dados
  • Revogar consentimento

8. Contato

Para questões sobre privacidade, entre em contato pelo nosso canal no Telegram.
